Introducción a las series de tiempo

Definición

Una serie de tiempo es una serie de datos ordenados en orden cronológico.

Ejemplos

Modelos de series de tiempo básicos

  • Ruido blanco (White Noise (WN))
  • Caminata aleatoria (Random Walk (RW))
  • Modelo autoregresivo (Autoregression (AR))
  • Media móvil (Simple Moving Average (MA))

Supuestos básico

  • Las observaciones consecutivas tienen la misma separación.
  • Se utiliza métodos de medición discretas.
start(AirPassengers)
## [1] 1949    1
frequency(AirPassengers)
## [1] 12
deltat(AirPassengers)
## [1] 0.08333333

Identificando tendencias

El determinar la tendencia de una serie nos puede permitir calcular valores perdidos o predecir posibles valores que puede tomar la serie de tiempo.

Identificando tendencias

Ruido blanco

El ruido blanco es el proceso estacionario más simple.

Una serie \(Y_1, Y_2, \ldots\) es un ruido blanco débil de media \(\mu\) y varianza \(\sigma^2\), es decir, \(WN(\mu, \sigma^2)\) si:

  • \(\mathbb{E}(Y_t) = 0\)
  • \(Var(Y_t) = \sigma^2\)
  • \(Cov(Y_t, Y_s) \neq 0\) para todo \(s\neq t\)

Ejemplo de ruido blanco

white_noise <- arima.sim(model = list(order = c(0, 0, 0)), n = 100)
ts.plot(white_noise)

Caminata aleatoria

Se define como: \[ Y_t = Y_{t-1} + \varepsilon_t\] donde \(\varepsilon_t\) es un ruido blanco.

O más en general, se puede considerar \[ Y_t = c+ Y_{t-1} + \varepsilon_t\] donde \(c \in \mathbb{R}\).

Ejemplo de caminata aleatoria

random_walk <- arima.sim(model = list(order = c(0, 1, 0)), n = 100)
random_walk_diff <- diff(random_walk) 
par(mfrow=c(1,2));ts.plot(random_walk_diff);ts.plot(random_walk)

Ejemplo de caminata aleatoria

wn_drift <- arima.sim(model=list(order=c(0,0,0)),  n = 100, mean = 0.4)
rw_drift <- cumsum(wn_drift)
plot.ts(cbind(wn_drift, rw_drift), main ="")
